# 行情地址说明

## HTTP接口地址

### 1、批量查询K线接口( [接口文档](https://docs.infoway.io/rest-api/http-endpoints/get-candles))

**股票K线查询接口（POST）**：

```
data.infoway.io/stock/v2/batch_kline
```

**加密货币K线查询接口（POST）**：

```
data.infoway.io/crypto/v2/batch_kline
```

**外汇商品期货K线查询接口（POST）**：

```
data.infoway.io/common/v2/batch_kline
```

### 2、批量查询成交明细接口( [接口文档](https://docs.infoway.io/rest-api/http-endpoints/get-trade))

**股票成交明细查询接口（GET）**：

```
data.infoway.io/stock/batch_trade/{codes}
```

**加密货币成交明细查询接口（GET）**：

```
data.infoway.io/crypto/batch_trade/{codes}
```

**外汇商品期货成交明细查询接口（GET）**：

```
data.infoway.io/common/batch_trade/{codes}
```

### 3、批量查询盘口接口( [接口文档](https://docs.infoway.io/rest-api/http-endpoints/get-depth))

**股票盘口查询接口（GET）**：

```
data.infoway.io/stock/batch_depth/{codes}
```

**加密货币盘口查询接口（GET）**：

```
data.infoway.io/crypto/batch_depth/{codes}
```

**外汇商品期货盘口查询接口（GET）**：

```
data.infoway.io/common/batch_depth/{codes}
```
